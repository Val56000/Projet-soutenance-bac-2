A propos de ce site



I. Présentation du contexte

Dans le cadre de l'examen, je réalise ce site web personnel qui a pour but de compléter la liste des compétences demandées.
Il présente la totalité de mes projets réalisés en stage.




II. Le projet

1. Le cahier des charges

Le besoin : Mettre en place un projet de site perso avec système de connexion qui permettra de compléter la liste des compétences attendues.
La cible : Le jury
Langages qui devront être utilisés pour la conception du site : HTML, CSS, Bootstrap, PHP, JavaScript
Maquettage : Réalisation avec Balsamiq
Schéma relationnel : Réalisation avec UMLet
IDE utilisé : Visual Studio Code pour réaliser le site en local


Missions principales :
1. Réaliser un schéma relationnel de la base de données
2. Dans MySql, créer la base de données, les tables et les relations avec l'interface web PhpMyAdmin
3. Réaliser un diagramme de cas d'utilisation
4. Réaliser une maquette du site
5. Construire les pages html (view) en respectant la maquette et y ajouter le contenu texte, image, formulaires ...
6. Gérer le style des pages avec CSS et Bootstrap
7. Créer une connexion à la base de donnée
8. Mettre en place un CRUD (create, read, update, delete) (models) avec des requêtes préparées contre les injections SQL
9. Sécuriser les variables avec la fonction htmlspecialchars de PHP contre les failles xss
10. Créer une session permettant de stocker des données utilisateurs
11. Hacher les mots de passes utilisateur avec la fonction sha1 de PHP
12. Créer des boîtes de dialogue avec JavaScript pour suppression du compte et déconnexion

2. Les langages utilisés

HTML signifie « HyperText Markup Language » qu'on peut traduire par « langage de balises pour l'hypertexte ». Il est utilisé afin de créer
et de représenter le contenu d'une page web et sa structure. C'est un langage de représentation.



CSS (Cascading Style Sheets) est un langage de mise en forme des documents. Il permet d'apporter un style au éléments de la page web
(taille de police, couleur, etc ...).



Bootstrap est un framework CSS. Grâce aux composants autonomes mis à disposition du framework, les développements sont facilités et vont
beaucoup plus vite que lorsque le développeur code avec la technologie d'origine... C'est pour cette raison que les développeurs préfèrent
l'usage de frameworks car ils sont plus efficaces et plus rapides d'utilisation.

Quant à Bootstrap, c'est un ensemble de fichiers CSS et JavaScript fonctionnant ensemble et qu’on va pouvoir
utiliser pour créer des design complexes de manière relativement simple.



JavaScript est un langage de programmation dynamique complet qui, appliqué à un document HTML, peut fournir une interactivité
dynamique sur les sites Web. C'est un langage essentiel des applications web.



PHP, pour Hypertext Preprocessor, désigne un langage de programmation utilisé principalement pour la conception de sites
web dynamiques et qui est executé côté serveur.

Info : le cryptage est une fonction bidirectionnelle qui inclut le cryptage et le décryptage tandis que le hachage
est une fonction à sens unique qui transforme un texte brut en un résumé unique qui est irréversible.



Utilisation de la fonction session_start pour démarrer une session. On stocke dans la variable de session le username et
le password de l'utilisateur.

Rappel d'une session : Une session en PHP correspond à une façon de stocker des données différentes
pour chaque utilisateur en utilisant un identifiant de session unique.



Ligne 56, on utilise la variable de session où est stocké le "username" de l'utilisateur.
On interroge la base de données avec une requête SELECT en lui demandant de nous récupérer le prénom où le username
est égal au "username" saisi dans le formulaire de connexion.



Lorsque l'utilisateur clique sur déconnexion, session_destroy() détruit toutes les données associées à la session
courante et une redirection est faite sur la page de connexion.
